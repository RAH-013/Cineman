# Manual de Usuario

Bienvenido a la guía visual de **Cineman**. Aquí aprenderás a interactuar con la plataforma desde la perspectiva de un cliente, gerente o administrador.

---

## Asistente Virtual (Cineman Chat)

Cineman incluye un asistente inteligente para mejorar tu experiencia:

- **Acceso**: Haz clic en el botón flotante en la esquina inferior derecha.
- **Consultas**: Puedes preguntar por películas, horarios, géneros o incluso pedir recomendaciones basadas en tu estado de ánimo.
- **Acciones Rápidas**: El chat permite comprar boletos o ver detalles de películas directamente desde la conversación.

---

## Flujo de Compra de Boletos

El proceso principal de la aplicación está diseñado para ser rápido y seguro:

### 1. Exploración de Cartelera

Al entrar al sitio, verás la lista de películas disponibles. Puedes hacer clic en cualquier poster para ver detalles como:

- Sinopsis.
- Clasificación.
- Duración.
- Géneros.

### 2. Inicio de Sesión

Para realizar una compra, **es obligatorio estar autenticado**:

- Si no tienes cuenta, ve a **Registro** e ingresa tus datos.
- Inicia sesión con tu correo y contraseña. El sistema te mantendrá conectado mediante un token seguro.

### 3. Selección de Función y Asientos

Una vez elegida la película:

- Selecciona una fecha y un horario de función.
- Se abrirá el **Mapa Interactivo de Sala**.
- **Leyenda:**
  - 🟢 **Verde/Libre**: Disponible para seleccionar.
  - 🟡 **Amarillo**: Asiento que acabas de seleccionar.
  - 🔴 **Rojo/Ocupado**: Ya reservado por otro usuario (bloqueado).

### 4. Confirmación de Pago

Revisa tu resumen (número de asientos, precio total). Al confirmar, el sistema generará un ticket digital único.

---

## Mi Perfil y Mis Boletos

En la esquina superior derecha, encontrarás tu panel personal:

- **Mis Tickets**: Aquí se almacena el historial de todas tus compras. Cada ticket muestra el código de referencia, la película, sala y asientos.
- **Editar Perfil**: Cambia tu información básica o imagen de perfil.

---

## Funciones Administrativas

Si tu cuenta tiene roles elevados, verás opciones adicionales:

### Managers (Gerentes)

- **Panel de Películas**: Permite agregar nuevos estrenos (subir imagenes, títulos, descripciones).
- **Programación**: Crear y editar horarios de funciones para las salas disponibles.

### Administradores (IT)

- **Monitoreo**: Acceso a los logs generados por el Watchdog.
- **Mantenimiento**: Botones para ejecutar backups de la base de datos directamente desde la web.
- **Usuarios**: Gestión y visualización de la lista de usuarios registrados.
