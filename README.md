# Proyecto CINEMAN

Proyecto final de la materia optativa: Administración de sistemas Linux. Dicho proyecto consiste en una sitio web simulando un cine real, en el, el usuario podra ver un catalogo de peliculas y seleccionar la de su preferencia, es importante aclarar que solo podra comprar boletos si esta registrado en el sistema. Todo esto usando el stack LAMP (Linux, Apache, mySQL, PHP). Desarrollado por el equipo 4.


## Variables de Entorno

Para correr esta API, se requiere definir las siguientes variables de entorno:

`BACK_URI`

`BACK_PORT`

`JWT_SECRET`

`MYSQL_DATABASE`
`MYSQL_USER`
`MYSQL_PASSWORD`
`MYSQL_HOST`
`MYSQL_PORT`

`GOOGLE_CLIENT_ID`
`GOOGLE_CLIENT_SECRET`
